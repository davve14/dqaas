# dqaas

# Pre-reqs
npm

sbt

# Installation
Extract package so that you have a dqaas-client folder and and dqaas-server folder
```bash
cd dqaas-client
```
Download dependencies
```bash
npm install
```
Start the node server
```bash
npm run dev
```
Start server side server
```bash
cd ../dqaas-server
```
Run with auto commit (~)
```bash
sbt "~run 9696"
```
